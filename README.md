# 60 Days Algorithm Challenge

> 60 days of consistent problem-solving to prepare for technical interviews and master algorithm design fundamentals.

## Why This Challenge?

Two main goals drive this journey:

### 1. Interview Preparation
- Solve problems that commonly appear in tech interviews
- Learn to communicate solutions clearly with time/space complexity analysis
- Build confidence for whiteboarding and coding assessments

### 2. Algorithm Design Fundamentals
- Move beyond "memorizing solutions" to understanding *why* algorithms work
- Learn which data structure or technique fits which problem pattern
- Develop intuition for optimizing brute-force solutions step by step
